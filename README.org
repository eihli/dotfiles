** Keyboard layout
You won't get very far typing on an unfamiliar keyboard.

#+begin_src sh
setxkbmap -variant norman -option lv3:ralt_alt # Or `-variant ,us`
xmodmap -e "keycode 66 = Escape Escape" -e "keycode 9 = Caps_Lock Caps_Lock"
#+end_src
